# Upgrading JFrog Platform

## 前置条件
1. 已安装Artifactory/Xray(单点)
2. Artifactory版本为7.x/Xray版本为3.x
3. 请尽可能使用原安装方式升级，如：安装时使用RPM方式，则升级时也使用RPM
